# Agentic Financial Console (v1.0-RC1)

This repository encompasses the core implementation of the Agentic Financial Console, a state-of-the-art backend processing engine using LangGraph, FastAPI, and Celery for resilient financial workflow orchestrations.
It features **Multi-File Batch Ingestion**, **Deterministic Python Adapters**, and **Targeted Multi-Modal receipt layout extraction (VLM) explicitly tailored for Expense/Treasury domains** via Gemini.

For production requirements and deep architectural specs, please refer to `/docs` (e.g. `02_아키텍처설계서.md`, `03_인터페이스_및_계약명세서.md`).
